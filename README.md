# 株式会社エクシス コーポレートサイト刷新案

デザインコンセプト: **「白図面と真鍮 (WHITE BLUEPRINT & BRASS)」**
設備図面の記号言語(表題欄・通り芯・線種)を、白基調×真鍮+紺のアクセントバンドで組んだ、明るく採用重視のリニューアル案です。AdobeStockで購入いただいた写真2点(施工管理/業務用空調点検)を使用しています。

## ファイル構成

```
エクシスHP刷新/
├── index.html      … トップページ(事業・会社概要・アクセス・問い合わせ)
├── recruit.html    … 採用特設ページ(職種・待遇・募集要項・FAQ・エントリー)
├── assets/
│   ├── style.css   … 共通スタイル
│   └── site.js     … 共通スクリプト(スクロール演出・カウンター・メニュー)
└── README.md       … このファイル
```

フレームワーク不使用の純粋な HTML/CSS/JS。ビルド不要で、フォルダごとサーバーにアップロードすれば動きます(Google Fonts を読み込むためインターネット接続が必要)。

## ローカルでの確認方法

`index.html` をダブルクリックで開くだけでも表示できます。

## 公開方法(例)

1. **今のサーバーにアップロード** — このフォルダの中身をそのまま FTP 等でアップロード
2. **無料ホスティング** — GitHub Pages / Netlify / Cloudflare Pages にフォルダをドラッグ&ドロップでも可

## 掲載情報について

- 掲載している数値・会社情報・採用条件は、**すべて現行サイト (www.ekusies-k.co.jp) に掲載されていた情報のみ**を使用しています(2026年7月時点)。捏造した実績・数値はありません。
- 「創業17年目」は設立2009年10月からの計算です。年が変わったら更新してください。

## 差し替え・追記が望ましい箇所

| 箇所 | 現状 | 推奨 |
|---|---|---|
| お問い合わせ | 電話 + メールリンク | 旧サイトにあった問い合わせフォームが必要なら formrun / Googleフォーム等の埋め込み、またはサーバー側のフォーム設置 |
| 写真 | AdobeStock 2点(assets/img/)を使用。元データ(AdobeStock_*.jpeg)はフォルダ内に保管(gitには含めない) | 実際の社員・現場写真が撮れたら差し替えるとさらに効果的 |
| 先輩の声 | 未掲載(ダミーを載せない方針) | 社員インタビューが取れたら recruit.html に追加推奨 |
| お知らせ | リニューアル告知1件のみ | index.html の REVISION セクションに行を追加して運用 |
| 英文社名 | フッターは「株式会社エクシス」のみ表記 | 正式英文社名が決まっていれば追記 |
| ファイルダウンロード | 未移植(協力会社向け安全書類・請求書) | 必要なら旧サイトのPDF/Excelを assets/ に置きリンクページを追加 |

## 運用メモ

- お知らせの追加: `index.html` 内の `<!-- お知らせ(REVISION) -->` ブロックに `.rev-row` を1行コピーして追記
- 採用条件の変更: `recruit.html` の募集要項テーブルと「数字で見る働き方」の両方を更新
- 色・フォントの変更: `assets/style.css` 冒頭の `:root` 変数を編集
